# Security Policy

Intent-Aware UI is currently a documentation-only research initiative. There is no shipped code, package or running service, so there is no attack surface to report against yet.

## Reporting a Vulnerability

Once implementations (prototypes, examples, or the `@intent-aware/core` package) exist, please report vulnerabilities privately through GitHub's [private vulnerability reporting](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing/privately-reporting-a-security-vulnerability) feature on this repository rather than opening a public issue.

Until then, if you believe something in this repository has security implications, feel free to open a GitHub Discussion.
